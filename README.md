# SOEN357_EVERYTHINGZIT

## How to run FullCircle

Follow these steps to run the project locally:

1. Clone the repository
```bash
git clone https://github.com/AdnaneBJA/SOEN357_EVERYTHINGZIT.git
cd SOEN357_EVERYTHINGZIT
```

2. Navigate to the client folder
```bash
cd client
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```

5. Open your browser and go to:
http://localhost:5173  

---

## 🛠 Requirements

- Node.js
- npm (comes with Node.js)
